Curso de Programación Web

La programación web es una disciplina que utiliza lenguajes como HTML, CSS y JavaScript para construir páginas y aplicaciones web interactivas. HTML (HyperText Markup Language) se usa para definir la estructura de las páginas web. Por ejemplo, los encabezados, párrafos e imágenes.

CSS (Cascading Style Sheets) se utiliza para estilizar la presentación de las páginas web, como colores, tipografías y diseños. JavaScript permite añadir interactividad, como formularios dinámicos, botones y animaciones.

Para mostrar una imagen en HTML, usamos la etiqueta <img>, mientras que para definir colores de fondo usamos la propiedad CSS background-color.
